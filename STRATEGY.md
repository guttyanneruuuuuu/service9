## 📈 グロース戦略 (実行ステップ)

### Phase 1: ローンチ後 0 → 1,000 ユーザー (最初の2週間)
1. 自分の宇宙を1個ガチで作る (将来の起業計画/夢など、バズりそうなテーマで)
2. **Xに投稿**: スクショ + 「これ俺の頭の中。みんなのも見たい」
3. 友達5人にDM送って使ってもらう
4. **教育系・自己啓発系のXアカウントに送る** (「東大志望が作った思考可視化ツール」というナラティブ)
5. note / Zennで「高校生が作った新SNS」記事を書く

### Phase 2: 1,000 → 10,000 (1ヶ月)
1. **TikTok/YouTube Shorts**で「3分で君の頭の中を可視化する方法」動画
2. **Product Hunt**にローンチ (英語版必須)
3. **HackerNews "Show HN"** に投稿 (英語、技術的な観点で)
4. 海外Twitter (X) のテック界隈・PKM (Personal Knowledge Management) 界隈に投下

### Phase 3: 10,000 → 100,000 (3ヶ月)
1. **Pro プラン** (¥500/月) を実装
   - 無制限ノード
   - カスタム背景・テーマ
   - 独自ドメイン宇宙
   - 思考パターン分析
2. **企業向け Embed API** をBeta開始 (ブログに自分の宇宙を貼れる)
3. **メディア露出**: 「東大目指す高校生がSNS発明」の取材狙い

### Phase 4: 100,000 → イグジット候補 (6-12ヶ月)
- DAU 50K で Notion / Figma / Discord などから声がかかる規模
- $5M-$50M のExitか、シリーズAで自走か選べる位置に
- イグジット益 → ドラえもん開発資金
